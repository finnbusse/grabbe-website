## 2024-05-18 - Localized ARIA Labels for Icon-Only Buttons
**Learning:** This app uses German for UI text. When adding ARIA labels or titles to icon-only buttons (like delete or move actions), it is crucial to use the correct German terms (e.g., "Karte löschen", "Nach oben verschieben") inferred from the surrounding context to maintain a consistent language experience for screen reader users and tooltip viewers.
**Action:** Always check the surrounding text for the language used before adding accessibility labels, and ensure they match the application language.
